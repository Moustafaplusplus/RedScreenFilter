## Opening the Project in Android Studio

### Step 1: Open Android Studio
1. Launch Android Studio (version 2024.1+ recommended)
2. Click "Open" from the welcome screen
3. Navigate to: `/Users/moustafaothman/Desktop/RedScreenFilter/android`
4. Click "OK"

### Step 2: Gradle Sync
1. Android Studio will automatically detect the Gradle project
2. Click "Sync Now" if prompted
3. Wait for Gradle to download dependencies (~2-3 minutes first time)
4. If you see any SDK errors, install required SDK components when prompted

### Step 3: Setup Android SDK (if needed)
1. Go to: Tools > SDK Manager
2. Ensure you have:
   - Android SDK Platform 34 (Android 14)
   - Android SDK Build-Tools 34.0.0+
   - Android SDK Platform-Tools
3. Click "Apply" to install missing components

### Step 4: Run the App
1. Connect an Android device OR start an emulator:
   - Device Manager > Create Device > Pick any device (Pixel 6 recommended)
   - API 28+ emulator image
2. Click the green "Run" button (▶️)
3. Select your device/emulator
4. App should launch showing "Red Screen Filter" text

## Troubleshooting:

**"Gradle sync failed"**
- Click "File > Invalidate Caches > Invalidate and Restart"
- Ensure JDK 17 is selected in File > Project Structure > SDK Location

**"Android SDK not found"**
- Download Android SDK via Android Studio SDK Manager
- Update local.properties with correct sdk.dir path

**"Emulator won't start"**
- Enable virtualization in BIOS (VT-x for Intel, AMD-V for AMD)
- macOS: System Preferences > Security & Privacy > Allow "Android Emulator"
