## Enable hosting (one-time)

1. Push this repo to GitHub (`main` or `master` branch).
2. Repository **Settings → Pages → Build and deployment**: Source = **GitHub Actions**.
3. Push changes under `docs/` — the workflow `.github/workflows/pages.yml` deploys automatically.

Use the Privacy Policy URL in **App Store Connect** and **Google Play Console**.
